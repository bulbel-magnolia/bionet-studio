关于

原理与说明

它在做什么

这是为「基因人工神经网络」水质传感线路打造的可配置仿真台。把工程菌当成一颗会算账的神经元:多种污染物浓度作为输入,经基因线路加权、积分,最终用绿 / 黄 / 红三色荧光报告水质安全 / 预警 / 重污染,并具备「越线即锁」的双稳态记忆。本通用平台允许自由增减输入、改每路权重与 Km、调阈值与动力学参数 —— 默认 3 输入配置即三污染物传感器的基准线路。

数学链路

① Hill 编码:每路浓度 Pi 经启动子非线性响应映射为 0–1 信号

xi = Pin / (Kmin + Pin)

② 加权求和(突触权重):

Z_input = Σ wi · xi

③ 中间蛋白动力学(欧拉法积分):

dZ/dt = α · Z_input − β · Z   ⇒   稳态 Z_ss = (α/β) · Z_input

④ 三级判定:Z < θ1 绿(安全)/ θ1 ≤ Z < θ2 黄(预警)/ Z ≥ θ2 红(重污染)。

⑤ 双稳态记忆:Z 一旦越过 θ2 即锁定为红,即使污染物回落仍判重污染,直至手动重置。

默认参数

n=2;3 输入权重 w=[0.96, 0.72, 1.20](= 基准设定 wp=0.8 × wl=[1.2, 0.9, 1.5]);Km=[0.5, 0.3, 0.7];θ1=0.30、θ2=0.70;α=0.50、β=0.20;噪声 关。模拟 0–24 h,步长 0.1 h。
