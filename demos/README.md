# iGEM 软件组 · 通用平台「ASSAY 信号控制台」

围绕「基因人工神经网络」水质检测系统做的**可配置仿真平台**——一个自包含的单文件网页工具,作为 Best Software Tool 的框架雏形。它把建模组的数学模型做成「可自由配置网络、实时出图」的交互工具,按台式分析仪(benchtop analyzer)的观感设计:数字是主角,判定像仪表锁存,曲线像示波器扫描。

> 演进说明:早期并排演示用的 demo1(模型仿真界面)、demo2(荧光识别 App)已移除,项目统一以本「通用平台」为框架继续推进。仿真内核、设计系统、外壳与原先三件套一脉相承。

## 怎么打开

`demo3-通用平台/index.html` —— **一个自包含的 `.html` 文件,双击在浏览器打开即可,零配置、零依赖、完全离线**(无任何 CDN / 联网需求,连字体都以 base64 内嵌)。顶栏可一键切换**浅色「实验台」/ 深色「控制台」**主题(选择会被记住)。建议用 Chrome / Edge / Firefox。

## 这个平台做什么

把工程菌建模为一个「求和神经元」:1~5 路污染物浓度经 Hill 编码 → 加权求和 → 一阶动力学积分,再三级判定为绿/黄/红,并带「越过 θ₂ 即锁定」的双稳态记忆。

1. 输入个数 1~5 可增减,每路有自己的浓度滑块、权重、Km;节点参数 θ1/θ2/α/β/噪声、Hill n 可配置。
2. **可视化网络原理图**:输入块 → 权重/Km 连线 → Σ 求和神经元 → 三级输出,带信号流动画(小圆点沿连线流动,大小随该路贡献);悬停高亮、点击聚焦到对应配置行;浅/深主题下均完整换色(深色为发光接线图)。
3. **信号判定仪表**(LED 判定词 + 大号 Z 读数 + θ1/θ2 三段轨 + 记忆锁定徽标)+ **检测孔发光** + **时间轴 24h 回放** + 两张**示波器级曲线**(Z 随时间、GFP/YFP/DsRed 三色荧光)+ **等式代入面板**。
4. **预设电路库**:内置「默认三污染 / 单输入开关 / 双输入协同 / 五输入多重污染」;支持保存配置、导出 / 导入 JSON。
5. **还原默认**:回到 3 输入 + 基准参数的三污染物传感器基准线路。
6. 导出 CSV / PNG(带 Toast);键盘可达性(焦点环、Esc 关闭、空格播放/暂停、`prefers-reduced-motion` 降级)。

## 设计语言:ASSAY · 台式信号控制台

整套界面按「刚开机的台式水质分析仪」来设计——数字是主角,判定像仪表锁存,曲线像示波器扫描。

1. **字体(全部 base64 内嵌,离线可用)**:Bricolage Grotesque 只用于大标题与判定词;Hanken Grotesk 用于一切 UI 与正文;Geist Mono 用于一切数字、单位、公式、坐标、读数(等宽 tabular,动画时不跳动)。中文回退系统黑体(微软雅黑 / 苹方)。
2. **双主题**:默认浅色「实验台」(适合投影);一键切换深色「控制台」(荧光与曲线发光,更具现场冲击力)。所有图表/原理图在切换时自动换色。
3. **信号判定仪表(signal-meter)**:一个 LED 方块显示判定词(安全/预警/重污染 + SAFE/WARN/CRIT)、一个大号 Z 读数、一条按 θ1/θ2 分三段的轨道,游标针随 Z 平滑滑动;越过锁定阈值后游标冻结并亮起「记忆锁定」徽标(内嵌 SVG 锁,非 emoji)。
4. **检测孔(well)**:俯视发光孔——色相随三色荧光占比、亮度随总强度,锁定时显示锁标。
5. **示波器级曲线**:细网格、虚线阈值线(带 θ₁=/θ₂= 标签)、曲线下渐变填充、深色下的辉光描边、随播放移动的时间游标与坐标药丸。
6. **克制的色彩**:绿/黄/红只作指示色(LED、游标、圆点、辉光),不铺成背景;品牌青绿色只用于结构性强调。

## 数学模型(已核验)

```
输入浓度 P_i (0~1)
  → Hill 编码   x_i = P_i^n / (Km_i^n + P_i^n)        [n=2, Km=[0.5,0.3,0.7], 可选高斯噪声]
  → 加权求和    Z_input = Σ (wp·wl_i)·x_i             [默认 wp=0.8, wl=[1.2,0.9,1.5];通用平台里以每路有效权重 wp·wl 直接配置]
  → 中间蛋白    dZ/dt = α·Z_input − β·Z               [α=0.5, β=0.2, 欧拉法, 0~24h, dt=0.1, Z0=0]
  → 三级判定    Z<θ1 绿 / θ1≤Z<θ2 黄 / Z≥θ2 红        [θ1=0.3, θ2=0.7]
  → 双稳态记忆  Z 一旦越过 θ2 即锁红,直到手动重置
```

> 通用平台与原 DEMO 1 在模型层逐点等价:内核 `kernel.js` 一字未改;区别仅在通用平台把每路「权重」直接当作有效权重(固定 wp=1),默认值 [0.96,0.72,1.20] = 0.8×[1.2,0.9,1.5] 复现原三污染传感器。

## 工程决定

1. **图表用原生 canvas 手绘,不引 CDN 图表库**——完全离线、零依赖,双击即开,会上没网也稳。
2. **字体以 base64 内嵌**——为兼顾「离线单文件」与「不落入 AI 通用观感」,把 Bricolage Grotesque / Hanken Grotesk / Geist Mono 三款字体内嵌进文件,Latin 用内嵌字体、中文回退系统黑体。
3. **`_shared/` 是唯一真源**——共用 CSS/图表逐字符内联进 `index.html`;改了 `_shared/` 后运行 `node _shared/build.mjs` 重新内联,局部样式/脚本会被保留。

## 已知限制

1. **存档持久化**:配置存档用 localStorage,`file://` 下部分浏览器禁用,此时按钮自动禁用并提示改用「导出 / 导入 JSON」。
2. **PNG 导出**仅导出 Z 曲线图(图 A);荧光曲线与网络原理图未作图片导出。
3. **iGEM 合规尚未在仓库落地**:页脚声明「OSI 协议开源 + 托管 iGEM 官方仓库」,但仓库内目前**没有 LICENSE / 仓库 URL / 版权头**,需尽早补齐(见下)。
